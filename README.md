# Usos de git
## Nuevas ramas:
1. Para crear una nueva rama y cambiar a esta, se utuliza el comando: `git checkout -b` y a continuacion el nombre de rama.
  * Ej: **git checkout -b name_rama**.
2. Para crear una nueva rama y mantenernos en la rama actual se utiliza el comando `git branch` y a continuacion el nomre de la rama.
  *Ej: **git branch name_rama**.
